🏦 Bank Account System using OOP in Python

📌 Description
This project is a simple Bank Account System implemented using Object-Oriented Programming (OOP) concepts in Python.  
It demonstrates how classes, inheritance, and methods can be used to simulate basic banking operations.

 🎯 Features
- Create a bank account
- Deposit money
- Withdraw money
- Check insufficient balance
- Apply interest (for savings account)
- Display final balance

🧠 Concepts Used
- Classes and Objects
- Inheritance
- Constructors (`__init__`)
- Method Overriding
- Encapsulation

 🏗️ Project Structure
- `bankAccount` (Base Class)
  - Attributes: account number, balance
  - Methods: deposit(), withdraw()

- `savingAccount` (Derived Class)
  - Inherits from bankAccount
  - Additional attribute: interest rate
  - Method: apply_interest()

 🛠️ Technologies Used
- Python (Core Python)
- OOP Concepts

▶️ How to Run

2. Download or clone the repository
3. Open terminal and navigate to the project folder
